# 😈 SAMP AI Assistant
> First AI assistant inside SAMP (MoonLoader + Node.js)
---
# 🔥 Features
* 🤖 AI answers in-game (`/ai`)
* 🧠 Smart replies (not just echo)
* 🎮 Works directly in SAMP
* ⚡ Easy setup
---
## 🚀 Demo
```
/ai how to earn money
```
➡️ AI: "Try trucker job or mining "

---
## ⚙️ Installation
### 1. Install dependencies

```
npm install
```

### 2. Run server

```
node server.js
```

### 3. Put Lua script into:

```
moonloader/
```

---

## 🧠 Tech

* Node.js
* Express
* MoonLoader (Lua)

---

## 💡 Idea
This project adds a real AI assistant into SAMP to help players, increase retention and improve gameplay experience.
