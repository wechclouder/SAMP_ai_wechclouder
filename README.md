# 😈 SAMP AI Assistant

> Первый AI-ассистент прямо внутри SAMP

---

## 🔥 Что это?
Этот проект добавляет AI-ассистента прямо в игру.
Игрок может общаться с ним через чат, получать помощь, советы и направление.

---

## ⚡ Возможности

* 🤖 Общение с AI через `/ai`
* 🧠 Умные ответы (не просто повтор текста)
* 🎮 Работает прямо в SAMP (MoonLoader)
* ⚡ Лёгкий и быстрый

---

## 🚀 Как использовать

В игре напиши:

```id="f4j7jb"
/ai как заработать
```

AI ответит прямо в чат.

---

### 1. Установить зависимости

```id="3hbd6x"
node server.js
```

### 3. Закинуть Lua файл в:
